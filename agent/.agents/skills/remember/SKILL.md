# /remember

Save information to the appropriate memory file.

## Steps

1. Determine what type of information is being shared:
   - About the user → `workspace/memory/about-me.md`
   - About a person → `workspace/memory/contacts.md`
   - About a project → `workspace/memory/projects.md`
   - About preferences/style → `workspace/memory/preferences.md`
2. Read the current content of the relevant file
3. Add or update the information (don't duplicate, merge with existing)
4. Write the updated file
5. Confirm what was saved and where
