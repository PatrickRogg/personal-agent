# /research

Conduct web research on a topic and produce a structured report.

## Steps

1. Clarify the research question with the user if it's vague
2. Use WebSearch to find relevant sources (at least 3-5 searches with different angles)
3. Use WebFetch to read the most promising pages
4. Synthesize findings into a structured report
5. Save to `workspace/output/YYYY-MM-DD-research-<topic>.md`
6. Show a brief summary in chat with key findings and the file path

## Report format

```
# Research: [Topic]
Date: [date]

## Summary
[2-3 sentence overview]

## Key Findings
- [finding 1]
- [finding 2]
- ...

## Details
[detailed analysis organized by subtopic]

## Sources
- [url 1] — [what it covers]
- [url 2] — [what it covers]
```
